# Глоссарий предметной области

Данный словарь предназначен для фиксации общей терминологии, используемой в приложении:

* **Ипподром** — комплекс сооружений для проведения испытаний рысистых и скаковых лошадей и соревнований по конному спорту.
* **Управление (Администратор)** — сотрудник ипподрома, обладающий полными правами доступа к системе для внесения данных о забегах, участниках и расчета призовых.
* **Жокей** — профессиональный всадник, состоящий в штате ипподрома, работающий с различными лошадьми в состязаниях.
* **Владелец лошади** — физическое или юридическое лицо, заявляющее своих лошадей на соревнования.
* **Забег** — спортивное состязание лошадей на скорость.
* **Призовой фонд** — сумма денег, распределяемая между участниками, занявшими призовые места в забеге (50% за 1-е место, 30% за 2-е, 20% за 3-е).
