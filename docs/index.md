# Lab #4 Documentation

# Система управления ипподромом

## Описание задачи
Данный проект представляет собой консольное приложение для автоматизации работы ипподрома. Система позволяет вести учет лошадей, жокеев, проведенных забегов, а также автоматически распределять призовой фонд между победителями. Доступ к функциям разделен между тремя ролями: Управление (Администратор), Жокей и Владелец лошади.
